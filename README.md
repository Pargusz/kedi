# 🐾 Üzüm - Merve'nin Kedisi

Merve için yapılmış sevimli bir sanal kedi web uygulaması.

## Özellikler
- 🍖 Besle, 💧 Su ver, 🎾 Yün top ile oyna, 🛁 Banyo yaptır, 😴 Uyut
- 💬 Her dakika Merve'ye tatlı iltifatlar (fizik temalı sürprizler dahil!)
- 🌙 Gündüz/Gece değişen gökyüzü
- 💾 Firebase + localStorage ile veri kalıcılığı
- 📱 Mobil uyumlu tasarım
